CS2640 Modern Storage Systems
Caching Competition

Assignment Overview

The performance of a storage system is often dictated by its caching strategy. An effective cache will significantly reduce data access latency and improve overall system throughput. The caching competition is an assignment that is designed to encourage students to learn and explore the design of cache eviction algorithms. Students will implement their own cache eviction algorithms and compete against each other based on the performance of their algorithms on a set of benchmark workloads.

Competition Details

  • Implementation: Students will implement their cache eviction algorithms in a provided framework. The framework will handle the basic cache operations, while students will focus on the eviction logic.
  • Workloads: A set of benchmark workloads will be provided to evaluate the performance of the cache algorithms. These workloads will simulate real-world access patterns.
  • Evaluation Metrics: The performance of each cache algorithm will be evaluated based on metrics such as hit rate, latency, and throughput.
  • Competition Format: The competition will be held in multiple rounds, with students submitting their implementations for evaluation after each round. The top-performing algorithms will be recognized at the end of the competition.
