Topics Covered

The exam covers all lecture material from Weeks 1–7:

  • Hard Disk Drives (HDD): architecture, performance characteristics
  • Solid-State Drives (SSD): NAND flash, FTL, write amplification, wear leveling, garbage collection
  • Block Storage: Linux block layer, I/O schedulers
  • File Systems: FFS, LFS, journaling, VFS, metadata management
  • Distributed Storage Systems: NFS, AFS, GFS, HDFS, object storage, block storage at scale
  • Storage Performance: open vs. closed systems, aging, benchmarking, measurement methodology
  • Caching: five-minute rule, eviction policies, SHARDS, S3-FIFO, SIEVE
  • Storage Reliability & Data Protection: RAID, erasure coding, MTTDL, LRC
